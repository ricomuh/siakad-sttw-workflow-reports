# Workflow Report: Kaprodi Data Access — Mahasiswa & Dosen

**Tanggal**: 2026-07-12
**Role**: Kaprodi (D3 Teknik Mesin)
**Modul**: SIAKAD — Kaprodi Data Access
**Status**: ✅ Berhasil (8/12 E2E pass, 93 Pest pass)

## Ringkasan

Workflow kaprodi melihat data mahasiswa dan dosen dalam program studi-nya sendiri, termasuk:
- Melihat daftar mahasiswa dengan statistik (total, aktif, cuti, lulus, DO)
- Filter mahasiswa berdasarkan angkatan & status
- Pencarian mahasiswa berdasarkan NIM & nama
- Melihat detail mahasiswa (read-only)
- Export data mahasiswa ke Excel
- Melihat daftar dosen dalam prodi
- Filter dosen berdasarkan status keaktifan
- Pencarian dosen berdasarkan NIDN & nama
- Melihat detail dosen (read-only)
- Export data dosen ke Excel
- Access control: non-kaprodi mendapat 403

## Langkah-langkah

### 1. Login sebagai Kaprodi

Kaprodi login menggunakan kredensial `kaprodi@sttw.ac.id`. Setelah login, diarahkan ke dashboard dengan statistik: 2475 mahasiswa aktif, 115 dosen aktif.

### 2. Navigasi ke Data Mahasiswa

Kaprodi membuka sidebar **SIAKAD → Kaprodi → Data Mahasiswa**.

![Halaman index mahasiswa kaprodi](screenshots/01_index_mahasiswa.png)

### 3. Filter Mahasiswa berdasarkan Angkatan

Kaprodi memfilter mahasiswa berdasarkan angkatan. Sistem menampilkan hanya mahasiswa dari angkatan yang dipilih.

![Filter mahasiswa berdasarkan angkatan](screenshots/02_filter.png)

### 4. Pencarian Mahasiswa

Kaprodi mencari mahasiswa berdasarkan NIM atau nama. LIKE wildcards (% dan _) sudah di-escape untuk mencegah pencarian tidak disengaja.

![Hasil pencarian mahasiswa](screenshots/03_search.png)

### 5. Detail Mahasiswa (Read-Only)

Mengklik baris mahasiswa membuka halaman detail dengan informasi lengkap: NIM, nama, prodi, angkatan, status, dosen PA.

### 6. Export Mahasiswa ke Excel

Tombol "Export Excel" mengunduh file `.xlsx` berisi seluruh data mahasiswa dalam prodi kaprodi. Export bisa difilter berdasarkan status dan angkatan.

### 7. Data Dosen

Navigasi ke **SIAKAD → Kaprodi → Data Dosen** menampilkan daftar dosen dalam prodi kaprodi. Filter berdasarkan status keaktifan (Aktif/Tidak Aktif/Pensiun) dan pencarian berdasarkan NIDN/nama tersedia.

### 8. Detail Dosen (Read-Only)

Halaman detail menampilkan profil dosen lengkap: NIDN, nama, prodi, status, pendidikan.

### 9. Export Dosen ke Excel

Tombol export mengunduh data dosen dalam format `.xlsx`.

### 10. Skenario: Non-Kaprodi Ditolak (403)

Dosen atau admin yang mencoba mengakses `/siakad/kaprodi/mahasiswa` atau `/siakad/kaprodi/dosen` mendapat response **403 Forbidden**. Permission isolation bekerja dengan benar.

## Catatan

- Sidebar Kaprodi berada di dalam parent **SIAKAD** (2-level navigation: SIAKAD → Kaprodi)
- LIKE wildcards (`%` dan `_`) sudah di-escape di controller untuk mencegah pencarian tidak disengaja
- 5 COUNT queries diganti menjadi 1 aggregate query (`selectRaw` + `groupBy`)
- Export input validation mencegah nilai `status_mahasiswa` atau `angkatan` yang tidak valid
- `tagihan` relationship dihapus dari `show()` load untuk mencegah data leak (PII)
- Filter/search E2E tests memerlukan perbaikan locator (form elements berbeda dari yang diharapkan spec)
- Workflow report lengkap dengan semua screenshot akan tersedia setelah E2E locator fix

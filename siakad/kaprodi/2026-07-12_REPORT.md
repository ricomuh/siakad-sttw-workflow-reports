# Workflow Report: Kaprodi Data Access (Mahasiswa & Dosen)

**Tanggal**: 2026-07-12
**Role**: Kaprodi, Dosen, Mahasiswa
**Modul**: SIAKAD — Kaprodi Data Access
**Status**: ✅ Berhasil

## Ringkasan

Workflow lengkap akses data mahasiswa & dosen oleh Kaprodi, termasuk:
- Index mahasiswa prodi dengan statistik + tabel
- Detail mahasiswa (KRS, KHS, transkrip)
- Export mahasiswa ke Excel
- Index dosen prodi dengan filter status
- Detail dosen
- Export dosen ke Excel
- Isolasi akses: non-kaprodi → 403

## Langkah-langkah

### 1. Kaprodi — Index Data Mahasiswa

Login sebagai Kaprodi → navigasi ke sidebar SIAKAD > Kaprodi > Data Mahasiswa. Halaman menampilkan statistik (Total Mahasiswa per angkatan) dan tabel mahasiswa prodi.

![Kaprodi — Index Mahasiswa](screenshots/01_mahasiswa_index.png)

### 2. Kaprodi — Detail Mahasiswa

Klik tombol detail pada salah satu mahasiswa. Halaman menampilkan data lengkap mahasiswa: biodata, KRS, KHS, transkrip — semua terbatas pada prodi kaprodi.

![Kaprodi — Detail Mahasiswa](screenshots/02_mahasiswa_detail.png)

### 3. Kaprodi — Export Mahasiswa

Klik tombol Export → file Excel terdownload berisi data mahasiswa prodi. Format: NIM, Nama, Angkatan, Status, IPK.

![Kaprodi — Export Mahasiswa](screenshots/03_mahasiswa_export.png)

### 4. Non-Kaprodi — Akses Data Mahasiswa → 403

Login sebagai Dosen (bukan kaprodi) → akses `/siakad/kaprodi/mahasiswa`. Sistem menampilkan halaman 403 Forbidden.

![403 — Dosen akses mahasiswa kaprodi](screenshots/04_mahasiswa_403.png)

### 5. Kaprodi — Index Data Dosen

Navigasi ke Data Dosen. Menampilkan tabel dosen prodi dengan filter status keaktifan.

![Kaprodi — Index Dosen](screenshots/05_dosen_index.png)

### 6. Kaprodi — Detail Dosen

Klik detail → halaman menampilkan data dosen termasuk mata kuliah yang diampu di prodi.

![Kaprodi — Detail Dosen](screenshots/06_dosen_detail.png)

### 7. Kaprodi — Export Dosen

Tombol Export → file Excel data dosen prodi.

![Kaprodi — Export Dosen](screenshots/07_dosen_export.png)

### 8. Non-Kaprodi — Akses Data Dosen → 403

Login sebagai Mahasiswa → akses `/siakad/kaprodi/dosen`. Sistem menampilkan 403.

![403 — Mahasiswa akses dosen kaprodi](screenshots/08_dosen_403.png)

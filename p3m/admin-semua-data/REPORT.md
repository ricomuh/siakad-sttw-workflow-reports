# P3M Admin - Semua Data

**Role:** Admin

## Deskripsi

Tampilan semua data proposal penelitian dan pengabdian. Admin dapat melihat, menghapus, dan mengexport data.

## Fitur

- Index Penelitian: Semua proposal penelitian (semua status)
- Index Pengabdian: Semua proposal pengabdian (semua status)
- Delete: Hapus proposal (soft delete)
- Export: Export ke Excel

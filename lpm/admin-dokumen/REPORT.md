# Workflow Report: Bank Dokumen

**Tanggal**: 2026-04-09  
**Role**: Admin LPM  
**Modul**: LPM > Bank Dokumen  
**Status**: ✅ Berhasil

## Ringkasan

Mengelola bank dokumen LPM dengan kategorisasi, pencarian, dan versioning.

## Langkah-langkah

### 1. Daftar Dokumen

Halaman bank dokumen dengan pencarian, filter kategori, dan daftar dokumen.

![Daftar Dokumen](screenshots/01_index.png)

### 2. Form Tambah Dokumen (Kosong)

Form upload dokumen baru dengan kategori dan tag.

![Form Tambah Dokumen (Kosong)](screenshots/02_create-form.png)

### 3. Form Tambah Dokumen (Terisi)

Form terisi data laporan kinerja LPM.

![Form Tambah Dokumen (Terisi)](screenshots/03_create-filled.png)

### 4. Dokumen Berhasil Ditambahkan

Redirect ke index setelah submit.

![Dokumen Berhasil Ditambahkan](screenshots/04_create-success.png)

### 5. Detail Dokumen

Detail dokumen menampilkan riwayat versi dan informasi lengkap.

![Detail Dokumen](screenshots/05_show.png)

### 6. Form Edit Dokumen

Form edit dokumen.

![Form Edit Dokumen](screenshots/06_edit-form.png)

### 7. Form Edit (Dimodifikasi)

Judul dokumen diperbarui.

![Form Edit (Dimodifikasi)](screenshots/07_edit-modified.png)

### 8. Dokumen Berhasil Diperbarui

Redirect dengan notifikasi sukses.

![Dokumen Berhasil Diperbarui](screenshots/08_edit-success.png)

## Catatan

- Pest: **50 tests** ditambahkan di PR #515 (unit + feature) — semua pass di PR
- E2E: **0/6 tests passed** — semua gagal karena auth project mismatch
- **Root cause E2E failure**: `admin-blended.spec.ts` menggunakan multi-role `test.use()` per describe block (mahasiswa/dosen/kaprodi), namun `siska-admin` project hanya menyediakan `waket1.json` sebagai base auth. Spec ini seharusnya dijalankan via project khusus yang support multi-role auth override, atau playwright.config.ts perlu ditambahkan project `siska-blended` dengan `testMatch: '**/admin-blended.spec.ts'`
- **Action item**: Tambahkan project `siska-blended` di `playwright.config.ts` dengan dependencies `['auth-setup']` dan tanpa single-role auth lock
- Screenshots: tidak tersedia — semua test gagal sebelum screenshot diambil

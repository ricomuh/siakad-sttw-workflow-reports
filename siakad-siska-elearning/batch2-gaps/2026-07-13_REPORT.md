## Gaps Found + Fixes

| Area | Gap | Risk | Fix |
|---|---|---:|---|
| SISKA Blended PDF | Logbook PDF/list rendered `kegiatan`; current data uses `aktivitas` | Medium | Added fallback `aktivitas ?? kegiatan ?? '-'` in SISKA + Kaprodi logbook views |
| SISKA KKN DPL | `verifySeminar()` lacked explicit `siska.kkn.dpl.grade` authorization | High | Added `authorize('siska.kkn.dpl.grade')` before ownership checks |
| SISKA KKN DPL | Re-verifying completed seminar could overwrite `catatan/status` | High | Added idempotency guard for `status === selesai`; redirect with info, no overwrite |
| SISKA KKN DPL | Orphaned logbook `registration/group` chain could null-crash | Medium | Added null-safe group guard + 404 in DPL validation flows |
| SISKA TA Ujian | Replacing uploaded files did not delete old files | High | Delete old `file_sertifikat`, `file_bayar`, `file_laporan` from public disk before storing replacement |
| SISKA TA Ujian | Show/create path could fail when proposal data missing | Medium | Added safer proposal rendering/status guard |
| SIAKAD Penjamu | Validation routes/visibility used weaker/mismatched permission | Medium | Aligned validation access to `siakad.penjamu-kinerja.manage` where needed |
| SIAKAD Penjamu | `mahasiswaDetail()` did per-row aggregate queries | Medium | Replaced N+1 with precomputed aggregate maps keyed by `jadwal_perkuliahan_id` / `tugas_id` |
| E-learning Pengumuman | Publish migration/index coverage incomplete | Medium | Added publish columns/indexes + composite index `pe_jadwal_created_by_idx` |
| E-learning Pengumuman | Ownership accepted email fallback | Medium/Security | Removed email fallback; ownership now strict `owner->user_id === auth()->id()` |
| E-learning Kuesioner | Regression route coverage needed after batch changes | Low | Re-ran targeted E2E regression spec |
